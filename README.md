# fuzzql-tool

 FuzzQL is a lightweight SQL injection fuzzing tool designed to test web form parameters using non-destructive payloads.

The tool sends payloads to a target endpoint and compares the server response against a baseline request to detect abnormal behavior that may indicate possible SQL injection vulnerabilities.

This project was built as part of learning web application security and vulnerability testing techniques.
